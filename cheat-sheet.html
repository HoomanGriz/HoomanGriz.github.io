Structured Query Language (SQL)



SQL Cheat Sheet

SQL (pronounced "Sequel") stands for Structured Query Language. It is written in an ANSI standard computer language and allows you to access and manipulate data in a database. Some popular databases that use SQL are Oracle, Sybase, Microsoft SQL Server, DB2, Access, MySQL, and PostgreSQL. Below you will find some basic commands to help you become a more efficient SQL user. Please note that the commands are in all caps for easier reading. SQL can recognize your commands in either case.

Database Manipulation

CREATE DATABASE database_name Creates a database ex. CREATE DATABASE My_Cheat_Sheet
DROP DATABASE database_name Deletes a database ex. DROP DATABASE My_Cheat_Sheet

Table Manipulation

CREATE TABLE "table_name" ("column_1" "data_type_for_column_1", "column_2" "data_type_for_column_2", ...) Creates a table in the database CREATE TABLE Any_Person (LastName varchar, FirstName varchar, Age int)
ALTER TABLE table_name ADD column_name datatype(*optional* max number of chars) Add columns in an existing table ALTER TABLE Any_Person ADD Address varchar(12)
ALTER TABLE table_name DROP column_name datatype Delete columns in an existing table ALTER TABLE Any_Person DROP Address varchar(12)
DROP TABLE table_name Delete a table DROP TABLE Any_Person

Index Manipulation

CREATE INDEX index_name ON table_name (column_name_1, column_name_2, ...) Create a simple index CREATE INDEX PersonIndex ON Any_Person (LastName, FirstName)
DROP INDEX table_name.index_name Delete an index DROP INDEX Any_Person.PersonIndex

Data Manipulation

INSERT INTO table_name VALUES (value_1, value_2,...) Insert new rows into a table INSERT INTO Any_Person VALUES("Wozniak", "Steve", 64) OR
INSERT INTO Any_Person(LastName, FirstName, Age) VALUES ("Wozniak", "Steve", 64)
DELETE FROM table_name WHERE column_name = some_value Delete rows in a table DELETE FROM Any_Person WHERE LastName = "Wozniak"
TRUNCATE TABLE table_name Deletes the data inside the table TRUNCATE TABLE Any_Person

Select

SELECT column_name(s) FROM table_name
SELECT * FROM table_name 		Select data from a table
Select all data from a table 		SELECT LastName, FirstName FROM Any_Person
SELECT * FROM Any_Person
SELECT column_name(s) FROM table_name WHERE column operator value AND column operator value OR column operator value... Selects data using different operators. You can use =, <>(not equal), >, <, >=, <=, BETWEEN, and LIKE (when looking for a pattern) SELECT * FROM Any_Person WHERE (FirstName="Alison" OR FirstName= "Ali") AND LastName="Cox"
SELECT * FROM Any_Person WHERE FirstName LIKE "Ali%"
SELECT column_1 FROM table_name ORDER BY column_name Selects data and orders it by the column. SELECT LastName, FirstName FROM Any_Person ORDER BY FirstName

Join

SELECT column_1, column_2 FROM first_table_name INNER JOIN second_table_name ON first_table_name.keyfield = second_table_name.keyfield The INNER JOIN will join two table based on a shared key and return matching rows for the request SELECT Employees.LastName, Any_Person.Age FROM Employees INNER JOIN Any_Person ON Employees.Employee_ID=Any_Person.ID
